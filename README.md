# SQL Study Repository

SQL 기초부터 실전까지 단계별로 학습한 내용을 정리한 저장소입니다.

---

## 📁 파일 구성

| 파일명 | 주제 | 설명 |
|--------|------|------|
| **01.DDL.sql** | DDL (Data Definition Language) | 데이터베이스, 테이블, 스키마 생성 및 수정 실습 |
| **02.DML.sql** | DML (Data Manipulation Language) | 데이터 삽입(INSERT), 수정(UPDATE), 삭제(DELETE) 등 조작 문법 |
| **03.WHERE.sql** | WHERE 조건 | 데이터 필터링 조건문 학습 |
| **04.ORDER_BY.sql** | ORDER BY | 정렬을 이용한 결과 데이터 순서 제어 |
| **05.AGG.sql** | Aggregate Functions | SUM, AVG, COUNT 등 집계 함수 실습 |
| **06.GROUP_BY.sql** | GROUP BY | 그룹화 및 그룹별 집계 |
| **07.CASE.sql** | CASE 문 | 조건에 따른 값 분기 처리 |
| **08.JOIN.sql** | JOIN | INNER, LEFT, RIGHT, FULL JOIN 등 테이블 결합 실습 |
| **09.SubQuery.sql** | Subquery | 서브쿼리를 활용한 복합 쿼리 작성 |
| **10.INDEX.sql** | INDEX | 인덱스의 개념 및 성능 최적화 실습 |
| **11.VIEW.sql** | VIEW | 뷰 생성과 관리 실습 |
| **12.Transaction.sql** | Transaction | 트랜잭션과 COMMIT / ROLLBACK 실습 |
